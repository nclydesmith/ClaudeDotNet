# OpenClaude .NET

A C# .NET implementation of OpenClaude, designed for high performance and native deployment.

## Features
- Fully native .NET AOT compiled CLI.
- High-performance streaming LLM integration.
- Rich terminal UI via Spectre.Console.
- Extensible Tool system.

## Setup
1. Ensure .NET 10 SDK is installed.
2. Build the solution:
   ```bash
   dotnet build dotnet/OpenClaude.sln
   ```
